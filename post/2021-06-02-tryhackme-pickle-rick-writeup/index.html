Ice1187 ,  June 2, 2021 • 2 min read

這是一個非常 basic 的題目，但一開始還是因為 wordlist 沒有對的字串，稍微卡了一下。: (

學到的東西

PT 如果網頁沒東西，nmap 沒有特別的 port，gobuster 也沒有找到東西的時候，可以試試看換個 wordlists 搜網頁，像是：

  • raft-<size>.txt
  • raft-<size>-files.txt
  • directory-list-2.3-<size>/txt

有 command execution 之後，一定要試試看 sudo、找 user 的 .ssh/ 能不能讀寫。
